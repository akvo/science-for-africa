# Agent Docs Master Map

This directory (`agent_docs/`) contains all artifacts generated during the BMAD lifecycle product development process.

## Living Documents
These documents represent the **project skeleton** (overall purpose, shared architecture, etc.). They are continually updated to reflect the current high-level state.
- [prd.md](prd.md) (Product Requirements Document)
- [architecture.md](architecture.md) (System Architecture)
- [user-guide.md](user-guide.md) (User Guide)

## Chronological Records
These documents trace the history of decisions and specific tasks over time. They are strictly append/create-only.
- `features/`: PRD deep dives for specific features or tasks.
  - [raw-source-findings.md](features/raw-source-findings.md) (Raw notes from Figma and Google Docs)
- `adrs/`: Architecture Decision Records.
- `stories/`: Developer-ready user stories.
- `sprint-plans/`: Iteration tracking.
